STEP 2 OF 4
🔖 Save as a Bookmark in Safari
1. Open any page in Safari (e.g. google.com)
2. Tap the Share icon (box with arrow) at the bottom
3. Tap "Add Bookmark"
4. Name it "→ Budget" and tap Save
5. Now tap the Bookmarks icon (book icon) → find "→ Budget"
6. Tap Edit → tap the bookmark → delete the URL entirely
7. Paste the code you copied in Step 1
8. Tap Done
STEP 3 OF 4
🛒 Use It on Any Amazon Order
1. Go to amazon.com in Safari
2. Tap Returns & Orders
3. Tap any order to open the detail page
4. Tap Bookmarks → "→ Budget"
5. Your budget app opens with the order pre-filled!
6. Check the details, pick category, tap Record Purchase
WHAT IT CAPTURES
✅ Automatically Reads
• Grand Total (including tax) — e.g. $9.44
• Order date — e.g. June 12, 2026
• Item name — e.g. Amazon Basics Disposable...
• Order number — added to description

You just confirm the category and tap Record. Done in 5 seconds.
